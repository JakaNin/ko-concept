# パターン4: 海辺の記憶

## コンセプト概要

**タイトル**: 海に還れなかった人形たちの家
**時代**: 昭和後期～現代（1970-現在、時代混合）
**トーン**: 怖い + 神秘的
**キーワード**: 海、溺死、漁村伝承、水、鏡、境界、人柱

## ストーリー背景

### 世界観
加太は古くから漁村として栄えた。海は恵みをもたらすが、時に命を奪う。この古民家は、海で亡くなった人々を悼む場所だった。遺族は故人の人形を作り、海へ流す「人形流し」の儀式を行っていた。しかし、ある事件をきっかけに儀式は途絶え、海に還れない人形たちがこの家に取り残された。人形たちは今も海を求め、彷徨い続けている。

### バックストーリー
- 加太の古い風習「人形流し」: 海難事故の犠牲者を悼む
- この家は儀式の前に人形を一時的に安置する場所
- 1975年、大きな台風で多数の漁師が犠牲に
- 遺族が作った人形が一度に大量に集まる
- しかし、海が荒れて人形を流せない日が続く
- 待っている間に、人形たちに「変化」が起き始める
- 鏡に映る人形の表情が変わる、濡れた跡が現れる
- 恐れた人々は儀式を放棄し、家を封鎖
- 50年近く経った今も、人形たちは海を待ち続ける
- 満潮時、家の中で波の音が聞こえる
- 来訪者は「新しい人柱」として認識される可能性

### 感情の核心
- **人形の視点**: 海への渇望、故人への未練、境界に閉じ込められた苦しみ
- **体験者の感情**: 海への畏怖、生と死の境界への不安、水への恐怖

## 空間活用（部屋ごとの役割）

### 玄関（導入：やや暗い）
**役割**: 漁師の家
- 漁具（網、浮き、錨）
- 古い船の模型
- 海難事故の新聞記事（黄ばんだ切り抜き）
- 床が少し湿っている（演出）

### 廊下（移行：暗い）
**役割**: 陸と海の境界
- 床に砂が散らばっている
- 壁に波の音が響く
- 照明が波のように揺らぐ
- 人形が列を作って「海の方向」へ向かっている
- 足跡が濡れている（錯覚演出）

### 和室2（縁側接続）（序盤：暗い）
**役割**: 遺族の祈りの部屋
- 人形と共に供えられた花、お香
- 遺影と人形が並べられている
- 縁側の向こうに「海が見える」錯覚（プロジェクション）
- 人形の衣服が濡れている
- 「早く海へ」と書かれた手紙

### 和室3（中盤：暗い）
**役割**: 人形流しの準備室
- 小舟の模型と人形
- 海に流す準備がされたまま止まっている
- 儀式の手順書（古文書風）
- 壁一面に海の写真（荒れた海）
- 波の音が最も大きい

### 洋室（転換：最暗部手前）
**役割**: 禁忌の部屋（人柱伝説）
- 海への怒りを持つ人形たち
- 「なぜ流してくれない」の落書き
- 鏡に映る人形が実物と違う表情
- 水が滴る音（実際に水滴が落ちる）
- 人柱伝説の絵巻（加太の古い風習）

### 風呂場（サブクライマックス）
**役割**: 水の侵食
- 浴槽に水が張られている（暗くて底が見えない）
- 水面に人形が浮かんでいる（一部だけ）
- 鏡に映る自分の後ろに「誰か」
- 水の音が反響
- **演出**: 体験者は外から覗くだけ（入らない）

### 寝室（クライマックス：最暗部）
**役割**: 海の底
- 部屋全体が青白く照らされる（水中表現）
- 人形が浮遊しているように配置（透明糸）
- 壁に波紋の影（プロジェクション）
- 中央に「海に還った者」の人形（安らかな表情）
- 窓の外に海が見える（プロジェクション、満潮の海）
- 体験者が入ると、波の音が一気に大きくなる

### キッチン（解放：明）
**役割**: 陸への帰還
- 乾いた空間
- 温かい飲み物
- 「お帰りなさい」のメッセージ
- 窓から実際の空（または明るい景色）
